Description

Conway's Game of Life is a simulation in which many squares called cells are placed among a grid and their "lives" are simulated with a set a rules determining if they live, die or even create more life. There are plenty of patterns and starting grids that can provide very interesting results, these can be seen in the "pattern" drop down.
